<h1>Millions</h1>
Mllions is a stock trading simulation game developed as the final assigment for IDATT2003 - Progamming 2. 
he application allows users to simulate stock market trading, manage a portfolio, and track financial growth over multiple trading weeks.

<h2>Features: </h2>
- Stock Trading: Buy and sell various stocks listed on the exchange. 
- Portfolio Management: Track your shares, strategize and make a million
- File Support: Load stock data and make the market your own 
- Player Progression: Track net worth and build your way from a Novice to a Speculator. 
- Graphical User Interface: A javaFX-based interface designed using the Model-View-Controller architectural pattern

<h2>Key Components: </h2>
- Exchange: The central marketplace for handling stocks and advancing to the next week.
- Transaction System: Utilizes specialized calculators to handle the different transactions. 
- Observer Pattern: Ensures the GUI updates in real-time when changes happen

<h2>Technical Execution: </h2>
- Java, programming language used for the assignment
- JavaFX, used for the graphical user interface
- Maven, for project configuration, dependency management and building
- JUnit, used for unit testing 
- Code style, adheres to Google Java Style and is verified with CheckStyle
