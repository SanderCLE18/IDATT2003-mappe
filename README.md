<h1>Millions</h1>
Mllions is a stock trading simulation game developed as the final assigment for IDATT2003 - Progamming 2. 
The application allows users to simulate stock market trading, manage a portfolio, and track financial growth over multiple trading weeks.

<h2>Features:</h2>
<ul>
  <li><strong>Stock Trading:</strong> Buy and sell various stocks listed on the exchange.</li>
  <li><strong>Portfolio Management:</strong> Track your shares, strategize and make a million.</li>
  <li><strong>File Support:</strong> Load stock data and make the market your own.</li>
  <li><strong>Player Progression:</strong> Track net worth and build your way from a Novice to a Speculator.</li>
  <li><strong>Graphical User Interface:</strong> A JavaFX-based interface designed using the Model-View-Controller (MVC) architectural pattern.</li>
</ul>

<h2>Key Components:</h2>
<ul>
  <li><strong>Exchange:</strong> The central marketplace for handling stocks and advancing to the next week.</li>
  <li><strong>Transaction System:</strong> Utilizes specialized calculators to handle the different transactions.</li>
  <li><strong>Observer Pattern:</strong> Ensures the GUI updates in real-time when changes happen.</li>
</ul>

<h2>Technical Execution:</h2>
<ul>
  <li><strong>Java:</strong> Programming language used for the assignment.</li>
  <li><strong>JavaFX:</strong> Used for the graphical user interface.</li>
  <li><strong>Maven:</strong> For project configuration, dependency management, and building.</li>
  <li><strong>JUnit:</strong> Used for unit testing.</li>
  <li><strong>Code Style:</strong> Adheres to Google Java Style and is verified with CheckStyle.</li>
</ul>
